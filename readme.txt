Converts images into WebP format

== Description ==
WP Media WebP Converter gives you the ability to seamlessly convert JPEG and PNG images into popular webp format with a single click. Not only that, it allows you to create new attachments for converted .webp files and even you can replace existing images with converted .webp files. It also gives you the opportunity to do lossless conversion as well as smaller size .webp images directly from the Settings page.

== Installation ==
Download from WordPress plugin repository and activate the plugin. That\'s it! You are ready to go...
